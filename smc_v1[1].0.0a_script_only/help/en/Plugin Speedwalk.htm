Plugin Speedwalk

Plugin is used for executing speedwalks in format like 5wn3suuuu. The main difference between SMC and other MUD clients is that they send all commands immidiatly, SMC sends path by parts (specified number of steps every second), and speedwalking can be interrupted at any time by user or one of key phrases from mud (you were attacked or you are stuck). I suppose you can remember situations where after sending 25w and got in ass after second step, you watched spam like "No way! You are fighting" and etc, loosing time of fight. Now forget about this. You can also set reserved combinations that shouldn't be treated as a speedwalk, for example "se" or "news".
To set aborting key phrases and reserved combinations edit speedwalk.php and set arrays $sw_reserved and $sw_abort. By default configured for Solace MUD.
Plugin gets number of steps in second from variable $speedwalk_stepsize, which you can set directly in game with command !var.
REQUIRES: defined prompt.

Exported commands:
  • speedwalk
Exported functons:
  • speedwalk_encode
  • speedwalk_decode

SPEEDWALK

Syntax: !speedwalk [on|off]
Turns on/off mode of speedwalking. If no parameters, then switches mode. Also aborts currently executing speedwalk if you turn off. 
Switch speedwalking
    !speedwalk
Abort speedwalk (with alias)
    !alias {stop} {!@speedwalk off#!@speedwalk on}
Make 3 steps in a second
    !var speedwalk_stepsize 3

speedwalk_encode($path)

Compresses path. 
Compress path to var
    !var $newpath speedwalk_encode("wwwnnunddsssse")
Compress from var to var
    !var $path speedwalk_encode($path)

speedwalk_decode($path)

Unpack path. 
    !var $newpath speedwalk_decode("2n3su")
    !var $path speedwalk_decode($path)
