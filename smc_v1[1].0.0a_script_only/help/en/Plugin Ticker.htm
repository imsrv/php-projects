Plugin Ticker

Plugin is used to track time events (ticks). Ticker can be restarted by key phrases coming from mud. To set them edit ticker.php and add to array $tick_clears desired lines. All Ticker options are stored in variables starting with "tick", that can be directly set when playing.

Exported commands:
  • ticksize
  • ticknotify
  • tick

TICKSIZE

Syntax: !ticksize [seconds]
Sets the number of seconds in tick. If no param, then shows current value. 
Set the tick size to 45 seconds
    !ticksize 45

TICKNOTIFY

Syntax: !ticknotify [n1 n2 n3 ...]
Sets times to tick, when you'll be notified. If no params, then shows current value. Notifications are emulated as lines from MUD, so they can activate your actions, triggers, etc. 
Notify on 10, 5 and 3 seconds to tick
    !ticknotify 10 5 3
Notify about tick (0 seconds)
    !ticknotify 0
Turn off notifications (set non-existing seconds)
    !ticknotify -1

TICK

Syntax: !tick [start|stop]
If start, then restart to count seconds to tick, stop - stop the counting. If no params, then show seconds to tick. 
Show seconds to tick
    !tick
Turn off ticker
    !tick stop
