1. Phrase Search (" ")
Putting quotes around a set of words will only find Resultss that match the words in that exact sequence.

Example: "business directory"
Results: business directory, US Business Directory, global business directory pages, etc.


2. Wildcard Matching (*)
Attaching a "*" to the right-hand side of a word will return left side partial matches.

Example: bu*
Results: bus, business, etc.


3. Document Section Restrictions
Attaching one the following operators to the front of a search word will restrict the search to a certain document sections.

or: - will restrict searches to document Organizations only
Example: or:yellowbuilder
co: - will restrict searches to document Countries only
Example: co:united states
st: - will restrict searches to document States only
Example: st:california
ci: - will restrict searches to document Cities only
Example: ci:san jose
ad: - will restrict searches to document Addresses only
Example: ad:
zc: - will restrict searches to document Zip Codes only
Example: zc:95110
pn: - will restrict searches to document Phone Numbers only
Example: pn:123 456 7890
fn: - will restrict searches to document Fax Numbers only
Example: fn:123 456 7890
es: - will restrict searches to document Established In only
Example: es:2001
ws: - will restrict searches to document Web Sites only
Example: ws:www.yellowbuilder.com
ca: - will restrict searches to document Categories only
Example: ca:arts
id: - will restrict searches to document ID Numbers only
Example: id:10


4. Combining the Syntax
You may combine any of the query syntax as long as the syntax is combined in the proper order.

Example 1: business directory
Example 2: or:business directory
Example 3: "or:business directory"
Example 4: or:business*
