Seir Anphin v1.1.1 Beta Readme

-Requires-

  1. PHP 4.0 or higher. Try to avoid 4.1.1, it was only meant as an April fools joke.
  2. MySQL 3.2x.

-Installation Instructions-
  1. Create a database. This typically entails sending an email to your webhost or creating it yourself with a utility they have provided for you.
  2. Open upload/lib/config.php from the zip and fill in your database connection details between the quotes.
  3. Upload the contents of "root" to a directory on your webserver, maintaining the directory structure of all contents.
  4. Set the permissions for lib/images/dock and lib/files/dock at 777. This may not be necessary on all servers.
  5. Go to http://yourdomain.com/install_v111.php and follow the instructions there.

Finally, be sure to change the Administrator password and delete install.php when you're finished with it.



-Upgrade Instructions-
  1. Delete all PHP files from the previous installation.
  2. Upload the contents of the"upload" folder and fill in your database connectivity details in lib/config.php.
  3. Important: If you want to keep the "default" template set you will have to export it under a different name and reimport it later. Unfortunately, this process may be imperfect. I hope I've fixed that bug in this version, sorry.
  4. Open up http://yourdomain.com/install_v111.php in your web browser and follow the instructions there.
