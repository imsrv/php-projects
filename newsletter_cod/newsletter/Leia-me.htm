Introdução:

Este sistema de newsletter foi desenvolvido pela Web-tools www.web-tools.kit.net , utilizando a tecnologia PHP / MySQL

Instalação:
A Instalação do sistema é muito simples.
1º Passo:
Antes de tudo você deve abrir o arquivo config.php dentro da pasta admin configurando os seus dados de conexao com o banco de dadod MySQL, Relembrando antes da instalação certifique-se de que o seu servidor rode o PHP/MySQL. Dentro do mesmo arquivo contem as variaveis $site , $autor_email e $email_admin.
A Variavel $site voce precisa colocar o endereço do seu site entre os parenteses. A Variavel $autor_email, voce deve colocar somente o NOME do seu site ou o seu Nick ou qualuer outro nome que queira clocar Exemplo :
Web-Tools. A Variavel $email_admin você deve colocar o seu e-mail ou o do seu site que sera usádo como remetente.

2º Passo:
Execute no seu browse (certifique-se de que voce esteja mandado os arquivos pro servidor ou FTP) o arquivo instalar.php , apos executar o arquivo dentro do arquivo tera 2 campos de formulários que deverao ser preenchidos com o usuario e senha que voce deseja usar para efetuar o login no administrador do sistema.

Pronto ! está instalado, efetue o login e pronto.



Recursos:
O Sistema de varios recursos , os recursos do sistema oferece que voce delete , cadastre , e modifique dados de usuarios, mande mensagens para e-mails individualmente ou para todos os emails cadastrados. e ainda tem um recurso muito intereçante onde na hora de cadastrar um novo usuario para utilizar o sistema voce pode definir ele como um usuario comum e um usuario administrador.



Usuário Normal: Para segurança do administrador o usuario normal so tem acesso as principais sessoes para colaborar no site do usuário.
Usuário adminstradorl: O usuario adminstrador tem acesso TOTAL de adicao e exclusao de usuarios e e-mails

Na hora de mandar as mensagens voce bote adicionar códigos htmls e para ajudar fizemos um pequeno menu de ajuda para as pessoas que nao saibam muito de HTML possam inserir imagens links quebra de linhas etc em sua noticia.
