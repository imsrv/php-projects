Manlix Site Grabber
О скрипте
Manlix Site Grabber - граббер, скрипт сам залазит на нужный Вам сайт и выдерает нужную информацию.
Возможности скрипта

  1. Выдерает информацию с чужих сайтов.
  2. Управление скриптом, через веб панель, защищена паролем.
  3. Корректная работа скрипта, при error_reporting=E_ALL
  4. Корректная работа скрипта, при register_globals=Off
Системные требования

  • Веб сервер
  • PHP (4.1.0 или выше)
Права доступа (CHMOD)
index.php 660

images		 770
images/*.* 660

inc		 770
inc/*.php 660
inc/*.php 660

Обратите внимание!

ЕСЛИ ВЫ ПОЛЬЗУЕТЕСЬ БЕСЛАТЫНМ ХОСТИНГОМ, то скорей всего CHMOD нужно ставить другой, обычно у таких хостеров есть свой F.A.Q. обязательно почитайте его, а то скажите, что мол скрипт не рабочий, а на самом деле окажется, что CHMOD не такой какой надо поставили.
Установка скрипта

НЕ УСТАНАВЛИВАЙТЕ СКРИПТ В ПАПКУ CGI-BIN
  1. Создайте на своём веб сервере папку grabber.
  2. Загрузите в неё файл index.php и две папки images & inc (вместе с их содержимом).
  3. Установите на файлы скрипта CHMOD (права доступа, установить их можно из почти любого ftp клиента).
  4. Откройте веб браузер, введите в нём: http://ВашСайт/grabber/index.php
  5. Если Вы не увидели перед собой никаких ошибок, значит скрипт функционирует правильно :)

Правильный вызов скрипта, через SSI
Сначала, выясните у своего хостинг-провайдера, в каких файлах выполняется ssi и выполняется ли оно вообще. По-умолчанию, ssi, выполняется в файлах с расширением *.shtm и *.shtml
(примеры имён таких файлов: index.shtm, grabber.shtml и т.д.)
Правильно SSI вставляется так:
<!--#include virtual="/grabber/index.php?$QUERY_STRING" -->

Например, есть файл grabber.shtml, он находится в папке /pub/home/www (полный путь), а сам скрипт Manlix Site Grabber, находится в /pub/home/www/grabber, а нужно вывести результат работы скрипта Manlix Site Grabber в файл grabber.shtml, делается это так: вставляем в сам файл grabber.shtml ssi тэг:

<!--#include virtual="/grabber/index.php?$QUERY_STRING" -->

Теперь откройте, через браузер файл grabber.shtml (обратитесь к своему сайту, примерно так: http://мой_сайт.ру/grabber.shtml).
Правильный вызов скрипта, через INCLUDE (PHP)
Например, нам нужно вставить результат выполнения скрипта в файл grabber.php (пускай для примера этот файл находится в папке /pub/home/www).

Откройте файл grabber.php впишите в него:

<? include("/pub/home/www/grabber/index.php"); ?>

Теперь откройте, через браузер файл grabber.php (обратитесь к своему сайту, примерно так: http://мой_сайт.ру/grabber.php).
