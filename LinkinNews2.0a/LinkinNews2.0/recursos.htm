Codigo
 Função
[#TITULO#] Se colocado no templete este codigo será subtituido pelo titulo da noticia.
[#NOTICIA#] Se colocado no templete o mesmo sera substituido pela noticia completa.
[#RESUMO_N#] Se colocado no templete o mesmo será substituido por um BREVE resumo da noticia criando assim um link para que o visitante opsa ver a noticia completa clicando em um link.
[#R_COMENTARIOS#] Se colocado no templete o mesmo será substituido por um recurso para que os visitantes possam comentar as noticias postadas.
[#VIEWS#] Se colocado no templete o mesmo será substituido pelo numero de vezes que a noticia foi vista.
[#HeadLines_BLANK#] Se colocado no template o sistema vai funcionar como um sistema de Head Lines onde irá aparecer somente o titulo da noticia onde este titulo terá um link para ver a noticia completa em outra janéla.
[#HeadLines_SELF#] Se colocado no template o sistema vai funcionar como um sistema de Head Lines onde irá aparecer somente o titulo da noticia onde este titulo terá um link para ver a noticia completa na MESMA janela.
[#D#] Se colocado em seu template o mesmo será substituido pela data da noticia no formato dd/mm/aaaa
[#D2#] Se colocado em seu template o mesmo será substituido pela data da noticia no formato [ dd/mm ]
[#TO_FRIEND#] Se por ventura o usuário adicionar esta tag em seu template um botão onde o visitante poderá indicar a noticia para um amigo, ea mesma será enviada para o e-mail no qual o visitante escolher.
