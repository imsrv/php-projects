Database Parameters
Database Host: MySQL host address - localhost is fine if you host with Seiretto

Database Name:

The database must already exist before installation
- this is the same as your username for FTP if you host with Seiretto

Database Username:

Username to access MySQL
- this is the same as your username for FTP if you host with Seiretto

Database Password:

 Password to access MySQL (this will also used as the initial admin login password).
Personalisation
Your Company Name: This will be used to personalise the pages on your members area.
Admin Folder name: You should enter a unique name more than 7 chars (use only letters and numbers, do not use spaces, and do not use your password).
Your email address: Used as the admin login username, to notify you when a new member joins, and as the initial contact email address.

When the form is complete click the button above.
If the details are entered correctly and a connection is made to your database the following tables will be created:

pama_admin
pama_members
pama_logins
