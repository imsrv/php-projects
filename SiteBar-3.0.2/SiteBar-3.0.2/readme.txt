System Requirements
===================

    Backend:
    - MySQL 3.23
    - PHP 4.06
    - For full functionality PHP with SMTP configured

    Browser: (full featured access):
    - "HTML 4.0 Transitional" compatible browser with DOM 2 support
      and innerHTML defined (non standard attribute)
    - Cookies ON, Javacript ON
    - Tested with Mozilla 1.5, Firebird 0.61, Internet Explorer 6.0, Opera 7.0

    Browser: (limited, read-only access):
    - Any HTML browser
    - Tested with Pocket Internet Explorer 2002, Netscape 4.8
