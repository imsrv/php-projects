Installation
============

Prerequisites:

Download and install MySQL.
Download and install Apache
Download and install PHP 4.1.0 or greater.

Then, do the following:

1. Unzip the zip file in some subdirectory (e.g. EasyGallery) under the root
directory on your server.

2. Create the database for your EasyGallery and populate it with the
tables included at the begining of the index.php file.

3. Open index.php file in some editor and edit preferences specific for your
server and database.

You are done. Point the browser to
http:\\your_domain.com\EasyGallery\index.php
and enjoy your EasyGallery.
