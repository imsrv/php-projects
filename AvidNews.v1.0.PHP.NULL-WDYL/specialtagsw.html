Special Tags
Throughout AvidNews software, there are special tags you may use to enhance your templates.
Some of these tags will only work in certain templates, so please take note, as to which templates they work with.

Tags to use directly in your Articles
These tags only work under your "Article Management" section. Please read the description of the tags, to better understand which template they will work in, and what they do.

%newpage%
(works in Article input box)
If a writer has written or is writing a long article, they can break their article up into seperate pages with this tag. Simply put, whenever they want to start a new page, they only need to type this tag into their article. If a writer has chosen to use this tag, it will display "Back" and "Next" links at the bottom of each page. Keep in mind however, that this will also break your Print Only version of the article, up into different pages as well.
