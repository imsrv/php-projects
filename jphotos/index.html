jPhotos - Photo gallery script

What is jPhotos?

jPhotos is PHP script which able simple managing photo gallery.

Features:

- Automatic displaying of directory
- Description of photos are in separately files (no SQL needed)
- Automatic generation of thumbnails (GD2 library required)
- Tree structure galleries
- EXIF

Deployment:

example - simple example script
edit foto_browse.htx, foto_detail.htx, style.css to change layout
edit config.php (directory structures, etc.)

Structure of gallery:

  • foto
    • thumb - automatic generated directory with thumnails (check permissions)
      • Photo1.jpg
      • Photo2.jpg
      • Photo3.jpg
    • desc - directory with descriptions
      • Photo1.jpg.txt
      • Photo2.jpg.txt
      • Photo3.jpg.txt
    • Hollyday photos - subgallery
    • Photo1.jpg
    • Photo2.jpg
    • Photo3.jpg
