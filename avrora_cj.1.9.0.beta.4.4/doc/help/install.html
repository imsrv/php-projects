1.   распаковать дистрибутив в директорию вэб сайта.
2.   выставить права на файлы и директории:
      /sys_log/   - 0777 (rwx-rwx-rwx)
      /thumb/      - 0777 (rwx-rwx-rwx)
      /.htaccess      - 0666 (rw--rw--rw-)
      /cj_config.php      - 0666 (rw--rw--rw-)
      /thumb/thumb.csv   - 0666 (rw--rw--rw-)
      /thumb/top.txt      - 0666 (rw--rw--rw-)

4.   отредактировать cj_config.php указав правильные данные для доступа к базе данных
      define('DB_HOST','localhost');   // хост базы данных
      define('DB_LOGIN','root');   // логин пользователя к базе данных
      define('DB_PASS','');      // пароль пользователя к базе данных
      define('DB_DEVICE','vadim_pornolom');   // имя базы данных
5.   запустить процес инсталляции используя http://www.your-server.com/install/
6.   удалить директорию install

Доступ к панели управления находится по адресу http://www.your-server.com/cj_adm.php
где дефолтным паролем является слово admin

Регистрация трейдеров происходит по адресу http://www.your-server.com/cj_wbm.php
главное не забудьте отредактировать темплейты /faces/tpl.register.* и указать там свои данные.

Версии авроры как правило делятся на несколько веток.
1.X.0   - новая ветка, но не проверенная в работе.
1.X.1   - стабильная ветка, проверенная в работе, иногда с мелкими багфиксами
1.X.X   - стабильная ветка, с некоторыми последующими изменениями.
