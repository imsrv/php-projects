Global Settings - этот пункт позволит вам указать общие настройки для вашего CJ.

IP Expire Hour: указывает, сколько часов IP адрес будет считатся уникальным.

Out % to content: указывает процент выхода на контент по умолчанию. Это значиние используется тогда, когда процент четко не задан в параметрах.

Trader Stats (Hour): указывает за сколько последних часов показывать статистику трейдера.

Return Back %*: указывает процент возврата трафика трейдерам.

Turbo Force: при излишке трафика идет попытка умного деления трафика между трейдерами. (Отменяет параметр Return Back %) Траффик распределяется между трейдерами, относительно их весового коэффициента.

Force Prio: приоритет форсов. Число от 50 до 300. Чем выше значение, тем больший приоритет имеют форсы.

Count MAX Click: число кликов которое засчитывается одному уникальному IP адресу.

Accept New Traders: включить прием новых трейдеров.

Generate Top: генерировать ли топ каждый час. Если вы не используете топ, то отключите генерацию, это даст вам экономию процессорного времени.

* - данный параметр не является глобальным, и может быть изменен в настройках отдельного трейдера.
