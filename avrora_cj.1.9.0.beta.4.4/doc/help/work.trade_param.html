grp - тип и группа трейдера. Генерируется только самим Avrora CJ при помощи пункта Link Generator

fk - первый клик. Указывает сколько первых кликов посетителя должны вести на трейд. Например fk=2 указывает, что первые два клика будут вести на контент. Применяется вместе с параметрами dburl, url

p - указывает процент, который должен отдаватся на контент. Например p=50. Применяется вместе с параметрами dburl, url. Если параметр p не указан, то он по умолчанию является равным тому, который введен в админке CJ.

url - выход на определенный линк. Например url=http://www.domain.com/gallery1.html

dburl - выход на линк из базы CJ. База линков находится в пункте DB Url List. Выход работает таким образом, что трейдер для себя получает каждый раз новый линк. Очень удобно например отдавать таким образом в качестве контента галлеры спонсоров. Используется как dburl=yes

tr - выход на определенного трейдера. Указывается домен трейдера, например tr=domain.com

Иногда возникает необходимость вести консольные трейд. При консольном трейде реферер как правило теряется, поэтому ваши трейдеры должны присылать вам серферов на следующий адрес: http://www.ваш-сидж.com/?trader_domain.com
