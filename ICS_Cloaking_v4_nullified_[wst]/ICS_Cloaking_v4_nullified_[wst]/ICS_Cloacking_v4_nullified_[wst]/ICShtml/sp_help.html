ICSspider Help

ICSspider Variables:
URL to spider
Enter a URL that you would like the spider to retrieve. Any URL you specify will be fetched and displayed on your site.
Using User Agent:
 Whatever is entered here as the user agent will be used by the ICSspider when retrieving the page from the URL specified above. For example; if you were to specify Googlebot as the user agent, then the webserver returning the page will show in it's logs a user agent of Googlebot requesting the URL specified above. This tool is intended as a means for you to test ICS by requesting URL's of cloaked pages. The cloaking options will have to be set to make UA STAGE3 Active otherwise the cloaked page will not be served because it will not match the required spider footprints of the other stages.
  • Notes:
    • The spider will retrieve pages from any URL you specify.
    • The Images will be broken if they have been specified as relative links on the page you have retrieved, because the page is displayed in the spider's URL not the original URL of the page this is normal and does not mean the images are broken.
    • The spider will use any User Agent you specify and present it to the server where the URL you specified resides.
    • Remember that the User Agent you specify will get written to the logs of the server hosting the URL to retireve.
