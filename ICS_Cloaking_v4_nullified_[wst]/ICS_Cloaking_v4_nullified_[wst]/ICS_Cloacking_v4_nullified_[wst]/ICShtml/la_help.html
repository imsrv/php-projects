Log Analyzer Help

Log Display And Admin:
Choose A Log File
Select a file from the drop down list to take action on.

  • The visitor.log file will contain hits from human visitors if you have enabled logging of human visits.
  • The spider.log file will contain hits from known spiders if you have enabled logging of spider visits.
  • The footprints.log file will contain hits from spiders matching a spider footprint but of unknown origin, if you have enabled logging of unknown spider visits.
  • There may be other entries if you have chosen to "Rotate Logs" at any time, these will be saved copies of logs with the date appended to the file name of when they were Rotated.
Action For This Log:
 The action chosen here will operate on the log file chosen above.
  • There are four possible choices:
    • Display - will diplay a formatted report including elements based on the checked fields.

    • Rotate Log - will clear the chosen log and copy it to a new log with the date appended.

    • Delete Log - will permanently delete the chosen log and it's contents.

    • Display Raw - will display the raw log entries without any formatting.
Hits Per Page:
 Set this to the number of hits you want to display per formatted page.

This option has no effect unless "Action For This Log" is set to Display.
Include User Agent In Report:
 Check this box if you want the User Agent displayed in the formatted report. If the User Agent is prefixed and suffixed by two **asterisks** then this hit would be matched by GE STAGE 1 when set to Active.

This option has no effect unless "Action For This Log" is set to Display.
Include Referrer In Report:
 Check this box if you want the Referrer information displayed in the formatted report. The referrer will be created as a hot link, so that you can click on it and be taken to the query used to find your site.

This option has no effect unless "Action For This Log" is set to Display.
Include IP Address In Report:
 Check this box if you want the IP Address displayed in the formatted report. If the IP Address is prefixed and suffixed by two **asterisks** then this hit would be matched by GE STAGE 1 when set to Active.

This option has no effect unless "Action For This Log" is set to Display.
Include Search Keywords In Report:
 Check this box if you want the Search Keywords used to find your site displayed in the formatted report.

This option has no effect unless "Action For This Log" is set to Display.
