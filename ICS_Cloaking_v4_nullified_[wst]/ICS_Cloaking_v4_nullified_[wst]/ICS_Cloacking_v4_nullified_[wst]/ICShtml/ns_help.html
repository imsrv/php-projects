NSlookup Help

NSlookup Variables:
IP Address
Enter the IP Address you want to have a host name returned for. The address should be entered in the format of 111.222.333.444 not all addresses will resolve into a hostname. Sometimes no host name is defined for an IP address or the IP address itself is unused and therefore no host name has been defined in DNS.
DNS Server:
 You may leave the DNS Server set to granitecity.com which is a public DNS available for anyone to use, or you may enter the IP address of your own, or your ISP's DNS server.
Look Up Entire Class "c":
 Check this box to lookup more than one address at once. The number returned depends on what results per page has been set to.
Results Per Page:
 Set this to the number of results to display per page when Look up entire class "c" has also been selected. NSlookup will return that many lookups and provide a link at the bottom of the result page to advance that many results again.
