Password Admin Help

Change Password:
UserID
The userid is statically set at Admin, it cannot be changed. It provides an additional security layer and enables future upgrades to ICS to make it changeable.
New Password
Enter the new password that will be used to authenticate yourself to ICS. The password can be any combination of letters and numbers of 1 to 16 characters in length and without any intervening spaces.
Retype New Password:
 Enter the same password as you specified in the New Password field, this is a precaution to ensure that a mistyped password does not become your new password.
