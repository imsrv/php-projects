Navigation Help

Select Interface:
Set/Change Site Global Variables
Choose this interface to set or change your site global variables (paths, e-mail address, and directory names). Also used to estalish your paths, directory names, and e-mail address the first time you install ICS.
Set Cloaking And Notification Options:
 Choose this interface to set or change the spider detection stages, notification, and logging options. Set which stages will be active for spider detection, and early visitor detection.
Cloaking A Page - Step By Step:
 Choose this interface to have site specific instrunctions returned to you on how to cloak a page with ICS.
IP Address And User Agent Admin:
 Choose this interface to add, delete, and modify IP Addresses and User Agents. Also use this interface to add an entirely new engine to ICS.
Display Current IP/UA List:
 Choose this interface to display the list of IP Addresses and User Agents that is currently being used by ICS.
View, Delete, Rotate Logs:
 Choose this interface to view, delete, rotate logs. You must have made the logging function "Active" for each of these logs to be available.
NS Host Lookup By IP:
 Choose this interface to find the host name for an IP Address or a range within that IP.
ICS Spider - Fetch URL:
 Choose this interface to enter a URL and any User Agent and the ICSspider will fetch the page using your specified User Agent.
Change Password:
 Choose this interface to change the current password to a new password.
