  Manual integration  
Example1:
Your form:
<form action="http://server1.com/send.pl" method=post>
<input type=text name=email>
<input type=submit value="Subscribe">
</form>

Needs to be changed to:
<form action="#server-url#subscribe.cgi" method=get>
<input type=text name=email> - original field
<input type=hidden name=mm_action value="http://server1.com/send.pl"> - after processing form will be post here
<input type=hidden name=mm_email value=email> - field name where will be typed e-mail
<input type=hidden name=mm_from value=form> - required param
<input type=hidden name=mm_id value=14> - subscription ID
<input type=submit>
</form>

How to add from your script:
perl sample:
#!/usr/bin/perl
print "Content-type: text/html\n\n";
`/usr/bin/perl subscribe.cgi "mm_email=some@email.com&mm_first_name=Here first name&mm_last_name=Here last name&mm_id=2" &`;
exit;
